# Minimalist Personal Expense Tracker

A lightweight personal finance dashboard built with Next.js and Tailwind CSS. Record daily expenses, explore category insights, export data, and stay mindful of spending trends.

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to explore the tracker.

### Production Build

```bash
npm run build
npm start
```

## Key Features

- Quick entry form with category presets and optional notes
- CSV import for rapid onboarding and export for backups
- Inline filters for timeframe, category, and keyword search
- Snapshot cards with total spend, largest expense, and last-month trends
- Category breakdown with visual progress indicators
- Local storage persistence for privacy-first tracking

## Tech Stack

- Next.js App Router
- React 18 with client components
- Tailwind CSS
- date-fns for friendly date utilities

Feel free to adapt the styling or extend the data model to match your budgeting workflow.
